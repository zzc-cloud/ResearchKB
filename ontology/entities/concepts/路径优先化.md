# 路径优先化

## Object semantics
- 在知识图谱推理中，根据候选路径的累积代价与未来代价对其重要性进行排序的机制，用于筛选最值得送入 LLM 的推理路径。

## 概念定义
- 该概念由 PathMind 明确提出，用于刻画候选推理路径的重要程度。

## 核心内涵
- 路径优先化不是简单最短路，而是同时考虑当前累积语义代价与面向目标答案的未来代价估计。

## 与其他概念的关系
- 它直接服务于重要推理路径的识别。
