# 路径优先化

## Object semantics
- 路径优先化是由 [[../papers/PathMind - A Retrieve-Prioritize-Reason Framework for Knowledge Graph Reasoning with Large Language Models|PathMind paper]] 提出的概念，用于在查询子图中识别更关键的推理路径。

## 概念定义
- [[../methods/PathMind|PathMind]] 用路径优先函数综合累计代价与未来代价，对候选路径进行排序。

## 核心内涵
- 它不是简单路径长度或随机抽样，而是面向推理目标的语义感知路径选择机制。

## 与其他概念的关系
- 它直接服务于重要推理路径的发现与暴露。
