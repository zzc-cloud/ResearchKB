# 重要推理路径

## Object semantics
- 重要推理路径是由 [[../papers/PathMind - A Retrieve-Prioritize-Reason Framework for Knowledge Graph Reasoning with Large Language Models|PathMind paper]] 强化定义的概念，表示最值得暴露给 LLM 的高价值推理链。

## 概念定义
- 在 [[../methods/PathMind|PathMind]] 中，重要推理路径是经过路径优先化筛选后用于指导 LLM 回答的路径集合。

## 核心内涵
- 它强调路径对答案生成的贡献差异，而非把所有候选路径等价处理。

## 与其他概念的关系
- 它由路径优先化机制识别出来，并作为后续推理输入。

## 相关任务 / 场景
