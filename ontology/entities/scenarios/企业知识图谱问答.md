# 企业知识图谱问答

## Object semantics
- 一种面向企业知识服务与决策支持的应用场景，使用知识图谱上的结构化检索、路径筛选与多跳推理来回答复杂问题。

## 场景描述
- 该场景强调把知识图谱推理能力用于企业环境中的复杂问答、答案解释与知识服务。

## 核心挑战
- 关键挑战包括企业知识异构性、路径噪声控制、推理可解释性与响应效率平衡。

- 路径优先排序
- 高价值推理路径筛选
